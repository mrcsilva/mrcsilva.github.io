Portfolio

Ecoforest Control

An app that allows it users to control the stoves made by Ecoforest. I built the app using the Flutter framework that helped me to quickly build an app for both Android and iOS. Each stove has an integrated computer that provides a webpage were we can control the stove. However, that interface is not very user friendly, so I decided to take advantage of the already available APIs and used them in the app.

ShareIO

Ever wondered how you could easily share files between devices and maintain privacy? Here's the app for you! ShareIO allows you to share files between mobile devices without the use of a middle server to deal with the file storage. All happens thanks to WebRTC that establishes a connection directly to the device you want to share your file with and sends it.
